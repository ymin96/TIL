# IDE shortcut

> eclipse, vs, intellij IDE 들의 기본적인 단축키에 대하여 작성한다.



### Eclipse

* 자동 정렬: Ctrl + Shift + F
  (Window - Preferences 창의 Java - Code Style - Formatter 에서 스타일 변경 가능)
* 행 주석: Ctrl + /
* 범위 주석: Ctrl + Shift + /
* 범위 주석 해제: Ctrl + Shift + \
* 자동 완성 기능: Ctrl + Space
* 자동 import: Ctrl + Shift + O

---



### Visual Studio

* 자동 정렬: Ctrl + K, F
* 범위 주석: Ctrl + K, C
* 범위 주석 해제: Ctrl + K, U
* 자동 완성: Ctrl + Space

---



### Intellij IDE

* 자동 정렬: Ctrl + Alt + L
* import 문 최적화: Ctrl + Alt + O
* 행 단위 주석: Ctrl + /
* 범위 주석 처리/해제: Ctrl + Shift + /
* 자동 완성: Ctrl + Space
